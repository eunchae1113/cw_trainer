📻 CW Trainer

웹 브라우저에서 바로 즐기는 레트로 감성의 모스 부호(CW) 트레이닝 웹 애플리케이션입니다.
아마추어 무선사 및 모스 부호 입문자를 위해 제작되었습니다.

## ✨ 주요 기능 (Features)
- Straight Key 모드: `Spacebar`를 활용한 전통적인 입력 방식 지원
- Paddle Key 모드: `Q` (단점/Dit)와 `W` (장점/Dah) 키를 활용한 패들 입력

## 🚀 사용 기술 (Tech Stack)
- HTML5, CSS3, Vanilla JavaScript (No Frameworks)
- Web Audio API
